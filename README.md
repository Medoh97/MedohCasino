# 🎰 MedohCasino

A free Android casino game — **Blackjack**, **Buster Blackjack**, **Wild Fives**, **Farkle**, and **Slots** — with chips to win and a big store of unlockable cosmetics (card fronts/backs, table backgrounds, dice, relics).

## 📥 Download & install

1. Go to **[Releases → Latest build](../../releases/latest)** and download **`app-debug.apk`**.
2. Open the file on your Android phone. If prompted, allow **"Install unknown apps"** for your browser/file manager.
3. Tap **Install**.

**To update:** download the newest `app-debug.apk` and install it over the top — your progress is kept.

> The version number is shown on the Start and Settings screens. Please make sure you're on the latest before reporting a bug.

## 🐞 Report a bug or suggest something

Open a **[New issue](../../issues/new/choose)** and fill in the **Bug report** form. For anything involving timing or animations (a hand resolving oddly, a button not responding), a **screen recording** is by far the most helpful thing you can include.
