# 🎰 MedohCasino

A free Android casino game played with **in-game chips only** — no real money. Eleven games, a deep cosmetics store, prestige, auto-play, and a global leaderboard.

## 🎮 Games
**Blackjack** · **Buster Blackjack** · **3 Card Poker** · **High Card Flush** · **Baccarat** · **Ultimate Texas Hold'em** · **Pai Gow Poker** · **Wild Fives** · **Farkle** · **Slots** · **Craps**

## ✨ Features
- **Online leaderboard** — pick a name and compete globally on chips, prestige, total rounds, per-game wins, and rare hands (Triple 7s, Royal Flushes, Slots jackpots, and more). Fully optional and login-free; your saves are never affected.
- **Prestige** — reset for a permanent, stacking payout bonus across every game, plus exclusive animated card sleeves.
- **Auto-play** — let most games play a sound strategy hand after hand, with session summaries and logs.
- **Tables** — climb from the Back Room to the Vault; each table sets the stakes.
- **Huge cosmetics store** — fully illustrated card decks (and matching backs), table backgrounds, dice, and more. Colour-blind 4-colour decks supported.
- **5 save slots**, lifetime stats and achievements per game, and a built-in update checker.

## 📥 Download & install
1. Go to **[Releases → Latest build](../../releases/latest)** and download the **`MedohCasino-1.0.<version>.apk`** file.
2. Open it on your Android phone. If prompted, allow **"Install unknown apps"** for your browser/file manager.
3. Tap **Install**.

**To update:** download the newest `MedohCasino-…apk` and install it over the top — your progress is kept.

> The version number is shown on the Start and Settings screens. Please make sure you're on the latest build before reporting a bug.
